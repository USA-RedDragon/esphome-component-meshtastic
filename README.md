# ESPHome Meshtastic

This project provides ESPHome support for Meshtastic devices, allowing you to integrate them into your smart home setup. It includes custom components and configurations to enable communication using the Meshtastic protocol on ESPHome devices running SX126x or SX127x LoRa radios.
